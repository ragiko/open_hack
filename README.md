# 作成するアプリの概要
## 名前：魚ガチャ（仮）

## 概要
- ガチャボタンを押すとランダムに魚が出てくる。魚を収集する。
## 機能
- ガチャ（複数　深海魚だけのガチャ＆普通のガチャ）
- 未収集魚表示
- ガチャ履歴表示
- 同じ魚を二度引くと、出世魚の場合だけ出世後の魚がGETできるという出世魚機能
- できれば作りたい
- 魚の画像も表示
- 魚の説明も表示

##現状
- 現段階ではガチャの動きはある程度実現できた。
